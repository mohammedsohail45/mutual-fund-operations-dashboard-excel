📊 Mutual Fund Operations Dashboard (Excel)

## 🚀 Project Overview
This project focuses on analyzing mutual fund transactions to monitor **Assets Under Management (AUM)**, transaction trends, and operational efficiency.  
The dashboard provides a clear and interactive view of investment patterns and processing performance.

---

🎯 Key Objectives
- Track total AUM across different fund types  
- Analyze transaction behavior (SIP, Lumpsum, Redemption)  
- Monitor operational efficiency (Success Rate & Processing Time)  
- Identify delays and failed transactions  

---
 🛠️ Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning  
- Charts & Visualization  
- KPI Metrics  

---
📈 Dashboard Features
- 📊 **AUM Analysis by Fund**
- 📌 **Transaction Breakdown (SIP, Lumpsum, Redemption)**
- 📉 **Monthly Investment Trend**
- 🔄 **Transaction Status Monitoring (Completed, Pending, Failed)**
- 📍 **Key Performance Indicators (KPIs)**
